Privacy Policy
==============

This application is an open-source project made available on the Google Play Store
solely for user convenience.

No Data Collection
------------------
This app does **not collect, store, process, or share** any personal or non-personal
data from its users.

Specifically:
- No personal information is collected
- No analytics or tracking technologies are used
- No third-party services that collect data are integrated
- No data is transmitted from the device
- No advertising or monetisation mechanisms are present

Permissions
-----------
The application does not require permissions that grant access to personal or
sensitive user data.

Open Source
-----------
The full source code of this application is publicly available and can be reviewed
by anyone. Users are encouraged to inspect the code to verify this policy.

Changes
-------
This Privacy Policy may be updated if the application changes in a way that affects
user privacy. Any changes will be reflected in this document.

Contact
-------
For questions or concerns regarding this Privacy Policy, please contact the developer
through the contact information provided on the Google Play Store or in this repository.
